    Veille pour la semaine du 4 mai 2020 au 10 mai 2020

    Une sélection de ce que j'ai écouté, vu ou lu et qui a marqué ma semaine (ou aurait dû) !

    Ecouté

    • J'ai écouté pas mal de podcasts intéressants sur Python cette semaine :

    • Episode #179 de PythonBytes avec comme invité Guido van Rossum himself ! Plusieurs pointeurs prometteurs

      • missingno

      • Messy datasets? Missing values? missingno provides a small toolset of flexible and easy-to-use missing data visualizations and utilities that allows you to get a quick visual summary of the completeness (or lack thereof) of your dataset.

      • LibCST

      • https://libcst.readthedocs.io/en/latest/

        • LibCST parses Python 3.0, 3.1, 3.3, 3.5, 3.6, 3.7 or 3.8 source code as a CST tree that keeps all formatting details (comments, whitespaces, parentheses, etc). It’s useful for building automated refactoring (codemod) applications and linters.

      • Une discussion sur l'indentation en Python : https://twitter.com/gvanrossum/status/1249549091584892928

      • Les espaces en Python ne sont significatifs qu'en début de ligne !

    • L'Episode #244 de Talk Python To Me date de fin 2019 mais les articles de Real Python qui y sont recommandés semblent intéressants.

    • Episode n°7 du Real Python Podcast sur Black et AsyncIO.

      • Il faudrait que je jette un oeil à EdgeDB
      • La thématique sur AsyncIO me semble intéressante à creuser même si je n'en suis pas encore là dans mon apprentissage de Python :
      • Les différentes vidéos sur AsyncIO de cette chaine YouTube mais également ces talks sur AsyncIO et la Musique et sur la manière de penser avec les Coroutines même si cette dernière est un peu ancienne.
      • Et il faudra que je jette un œil au formatteur de code Black.

    • Dans l'Episode n°6 du Real Python Podcast sur les API Rest et le Well-Grounded Python Developer, il y a plein de liens et pointeurs intéressants qu'il faudra que j'aille regarder plus en détail dès que je commencerai à développer avec Flask.

    • L'épisode #180 de PythonBytes donne plein de pointeurs intéressants, dont je retiens au moins les 3 pointeurs suivants :

      • Les nombres en Python
      • Codespell
      • Safer

    • L'épisode #6 du Real Python Podcast sur Python, La Data Science et Docker est une écoute intéressante quand on s'intéresse à ces sujets-là, avec pas mal de pointuers dans les shownotes.

    • La vision du futur de Python par plusieurs personnes de ThoughtWorks est toujours bonne à écouter.

    • Command-Line Heroes est un podcast vraiment agréable et intéressant sur l'histoire de l'informatique. Le récent épisode avec Steve Wozniak vaut l'écoute.

    • Le dernier épisode de CoRecursive avec Bruce Tate aborde l'apprentissage des langages de programmation et ce que cela apporte, notamment la joie qu'apporte la compréhension. Bruce Tate a également attisé ma curiosité à propos du langage Pony.

    Sur mes radars

    • J'aime bien ce tweet de Sandro Mancuso sur le pragmatisme

    • Pragmatism is not the opposite of excellence. It does not mean doing a substandard job. Pragmatism defines the boundaries (or scope) that excellence must exist within.

    • Je le mets volontiers en regard de ce tweet sur la TDD ou cet autre tweet sur la dette technique de Vincent Déniel
